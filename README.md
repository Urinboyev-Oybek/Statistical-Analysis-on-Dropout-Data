# 📊 Data Cleaning & Statistical Analysis Project

## 📌 Overview

This project focuses on **data cleaning** and **statistical analysis** using Python.
Raw data was processed, cleaned, and transformed into a structured format, followed by statistical exploration and analysis.

---

## ⚙️ Project Structure

```
├── Datacleaning.ipynb   # Data preprocessing and cleaning
├── Statanalysis.ipynb  # Statistical analysis and insights
└── README.md           # Project documentation
```

## 🧹 Data Cleaning

The `Datacleaning.ipynb` notebook includes:

* Handling missing values
* Removing duplicates
* Data formatting and type conversion
* Feature preparation

## 📈 Statistical Analysis
